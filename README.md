# Crystal Ceylon - Private Tours Sri Lanka

A modern travel website for Crystal Ceylon - your gateway to personalized private tours in Sri Lanka.

## Features

- Browse tour packages and destinations
- Custom package builder
- Activity and things to do recommendations
- Beautiful destinations showcase
- Contact form for inquiries
- Fully responsive design

## Tech Stack

- React 18 with TypeScript
- Vite
- Tailwind CSS
- shadcn/ui components
- React Router DOM
- React Hook Form
- Zod validation

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open http://localhost:8080 in your browser

## Build

To create a production build:

```bash
npm run build
```
